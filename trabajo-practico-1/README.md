Ejercicio 1 | archivo : tp1_ejercicio_1.py
    Se define la clase Vehiculo con sus atributos correspondientes y un metodo describir.
    Luego se generan 2 clases (Auto y Moto) que heredan de Vehiculo, pero con el metodo describir() que sobreescribe al del padre.
    Se instancian varios vehiculos dentro de una lista que luego se recorre para llamar el describir() de cada objeto.

    Para correr el ejercicio se debe realizar :
    python tp1_ejercicio_1.py
